## 🔍 Hybrid Search Implementation

### Function: hybrid_search()
```sql
CREATE OR REPLACE FUNCTION hybrid_search(
  query_text TEXT,
  query_embedding VECTOR(1536),
  match_count INT DEFAULT 10,
  vector_weight FLOAT DEFAULT 0.7,
  fts_weight FLOAT DEFAULT 0.3
)
RETURNS TABLE (
  id UUID,
  content TEXT,
  source_type TEXT,
  vector_score FLOAT,
  fts_score FLOAT,
  final_score FLOAT
)
```

### Algorithm
1. **Vector Search:** `ORDER BY embedding <=> query_embedding LIMIT match_count * 2`
2. **FTS Search:** `WHERE search_terms @@ plainto_tsquery('arabic', query_text)`
3. **RRF Fusion:** `score = 1/(k + rank_vector) + 1/(k + rank_fts)`
4. **Return Top-K**

## 🔧 RAG Pipeline

```
User Query
    ↓
[1] Arabic Normalization
    ↓
[2] Generate Embedding (text-embedding-3-small)
    ↓
[3] Hybrid Search (pgvector + FTS)
    ↓
[4] Rerank (Cohere / Cross-encoder)
    ↓
[5] Filter by Threshold (score > 0.7)
    ↓
[6] Build Context (Top-K chunks)
    ↓
[7] LLM Prompt (Kimi K2.5)
    ↓
[8] Parse Response + Citations
    ↓
Return to User
```

### Prompt Template
```
أنت مساعد قانوني متخصص في الأنظمة السعودية. أجب على السؤال التالي بناءً على 
النصوص القانونية المقدمة فقط. استشهد بالمواد القانونية بوضوح.

النصوص المرجعية:
{context}

السؤال: {query}

إرشادات:
1. إذا لم تجد إجابة في النصوص، قل "لا توجد معلومات كافية في المصادر المتاحة"
2. استشهد بالمواد القانونية برقمها واسم النظام
3. لا تُضف معلومات من خارج النصوص المقدمة
```
