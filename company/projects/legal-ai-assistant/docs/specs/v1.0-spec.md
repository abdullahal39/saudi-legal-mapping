## 🔍 Hybrid Search Engine

### الخوارزمية

```typescript
async function hybridSearch(query: string, options: SearchOptions): Promise<SearchResult[]> {
  // 1. Query Classification
  const queryType = classifyQuery(query);
  // 'article_number' | 'court_case' | 'general_question' | 'procedural'
  
  // 2. Arabic Normalization
  const normalizedQuery = normalizeArabic(query);
  
  // 3. Parallel Search
  const [vectorResults, ftsResults, keywordResults] = await Promise.all([
    vectorSearch(normalizedQuery, options),
    fullTextSearch(normalizedQuery, options),
    keywordSearch(query, options)  // original query for numbers
  ]);
  
  // 4. Reciprocal Rank Fusion
  const weights = getWeightsForQueryType(queryType);
  const fusedResults = reciprocalRankFusion(
    vectorResults, ftsResults, keywordResults,
    weights.vector, weights.fts, weights.keyword
  );
  
  // 5. Reranking
  const rerankedResults = await rerank(fusedResults, query);
  
  return rerankedResults.slice(0, options.topK || 10);
}
```

### Weights حسب نوع السؤال

| نوع السؤال | Vector | FTS | Keyword |
|------------|--------|-----|---------|
| article_number | 0.2 | 0.3 | 0.5 |
| court_case | 0.3 | 0.5 | 0.2 |
| general_question | 0.6 | 0.3 | 0.1 |
| procedural | 0.4 | 0.4 | 0.2 |

## 🧠 RAG Pipeline

### 1. Query Preprocessing
```typescript
function normalizeArabic(text: string): string {
  return text
    .replace(/[إأآا]/g, 'ا')           // Unify alef variants
    .replace(/ى/g, 'ي')                 // Alef maksura to ya
    .replace(/ؤ/g, 'و')                 // Hamza on waw
    .replace(/ئ/g, 'ي')                 // Hamza on ya
    .replace(/[َُِْ]/g, '')              // Remove tashkeel
    .replace(/[ٱ]/g, 'ا')               // Small alef
    .toLowerCase();
}
```

### 2. Query Classification
```typescript
function classifyQuery(query: string): QueryType {
  // Pattern: "المادة 53" | "مادة ١٢٣"
  if (/المادة\s+\d+/.test(query)) return 'article_number';
  
  // Pattern: "حكم" | "قرار" + "محكمة"
  if (/حكم|قرار.*محكمة/.test(query)) return 'court_case';
  
  // Pattern: "كيف" | "ما هي" | "ما هو"
  if (/كيف|ما\s+(هي|هو)/.test(query)) return 'general_question';
  
  return 'general_question';
}
```

### 3. Context Retrieval
```typescript
async function retrieveContext(query: string): Promise<Context[]> {
  const results = await hybridSearch(query, { topK: 5 });
  
  // Group by source type
  const articles = results.filter(r => r.source === 'articles');
  const precedents = results.filter(r => r.source === 'judicial_precedents');
  const tameems = results.filter(r => r.source === 'tameems');
  
  // Format for LLM
  return formatContext({ articles, precedents, tameems });
}
```

### 4. Answer Generation
```typescript
async function generateAnswer(query: string, context: Context[]): Promise<Answer> {
  const systemPrompt = `أنت مساعد قانوني متخصص. استند فقط على السياق المقدم.
قوانين الإجابة:
1. استشهد بالمصدر (المادة/القرار/التعميم)
2. إذا كانت المعلومة غير موجودة في السياق، قل "لا توجد معلومات كافية"
3. لا تخترع إجابات
4. اكتب باللغة العربية الفصحى`;

  const userPrompt = `السؤال: ${query}

السياق:
${formatContextForLLM(context)}`;

  const response = await kimi.chat.completions.create({
    model: 'kimi-k2.5',
    messages: [
      { role: 'system', content: systemPrompt },
      { role: 'user', content: userPrompt }
    ],
    temperature: 0.1
  });

  return {
    answer: response.choices[0].message.content,
    citations: extractCitations(context)
  };
}
```
